# Template Project

这是一个跨平台的 C++ 项目模板。

## 项目结构

```
templateproject/
├── src/          # 源代码
├── include/      # 头文件
├── tests/        # 测试文件
├── build/        # 构建目录
└── docs/         # 文档
```

## 构建要求

- CMake 3.15 或更高版本
- C++17 兼容的编译器
- Git

## 构建步骤

```bash
mkdir build
cd build
cmake ..
make
```
